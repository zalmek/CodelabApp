>Методические указания по выполнению задания для платформы Android</h1>  <section class="chapter"><h2 id="36073ec8_14089" data-toc="36073ec8_14089">План</h2><ol class="list _decimal" id="36073ec8_14093"><p>Добавление зависимостей</p></li></ol></section><section class="chapter"><h2 id="36073ec8_14094" data-toc="36073ec8_14094">Создание проекта</h2><p id="36073ec8_14095">Создаем новый проект в <code class="code" id="36073ec8_14096">Android Studio</code>. В качестве шаблона выбираем <code class="code" id="36073ec8_14097">Empty Activity</code>. <figure  id="36073ec8_14098"><img alt="img.png" title="img.png" src="images/1.png"  class="" width="901" height="654"/></figure></p><p id="36073ec8_14099">Далее выбираем имя для проекта и нажимае на кнопку <span class="control" id="36073ec8_14100"><span class="emphasis" id="36073ec8_14101">finish</span></span><figure  id="36073ec8_14102"><img alt="img.png" title="img.png" src="images/2.png"  class="" width="897" height="653"/></figure></p><p id="36073ec8_14103">Затем необходимо некоторое время подождать, когда все необходимые файлы для работы приложения будут созданы, а все необход имые зависимости будут загружены.</p></section><section class="chapter"><h2 id="36073ec8_14104" data-toc="36073ec8_14104">Структура проекта</h2><figure  id="36073ec8_14105"><img alt="img.png" title="img.png" src="images/3.png"  class="" width="662" height="884"/></figure><p id="36073ec8_14106">Проект для <code class="code" id="36073ec8_14107">Android</code> имеет следующую структуру:</p><ul class="list _ul" id="36073ec8_14108"    ><li class="list__item" id="36073ec8_14109"><p>Папка <code class="code" id="36073ec8_14110">manifests</code></p></li><li class="list__item" id="36073ec8_14111"><p>Папка <code class="code" id="36073ec8_14112">java</code></p></li><li class="list__item" id="36073ec8_14113"><p>Папка <code class="code" id="36073ec8_14114">res</code></p></li><li class="list__item" id="36073ec8_14115"><p><code class="code" id="36073ec8_14116">Gradle Scripts</code></p></li></ul><p id="36073ec8_14117">В папке <code class="code" id="36073ec8_14118">manifests</code> описываются файлы-манифесты, они декларируют особенности Android приложения.</p><p id="36073ec8_14119">В этих файлах описываются <span class="control" id="36073ec8_14120">основные компоненты</span>, которые будут использованы в приложении (всего их 4 - <code class="code" id="36073ec8_14121">Activity</code>, <code class="code" id="36073ec8_14122">Service</code>, <code class="code" id="36073ec8_14123">BroadcastReceiver</code>, <code class="code" id="36073ec8_14124">ContentProvider</code>), запрашиваемые приложением <span class="control" id="36073ec8_14125">разрешения</span> (например, доступ в Интернет), а также некоторые другие особенности работы приложения.</p><p id="36073ec8_14126">Манифест может быть <span class="emphasis" id="36073ec8_14127">только один</span> для конкретной сборки.</p><aside class="prompt" data-type="note" data-title="" id="36073ec8_14128"><p id="36073ec8_14129"><span class="control" id="36073ec8_14130">Важно</span></p><p id="36073ec8_14131">Некоторые, потенциально опасные разрешения, можно получить, только в процессе работы приложения, они называются <code class="code" id="36073ec8_14132">Runtime Permissions</code></p></aside><p id="36073ec8_14133">В папке <code class="code" id="36073ec8_14134">Java</code>, находятся 3 пакета, два из них предназначены для тестирования приложения, а в третьем будет описано само <span class="control" id="36073ec8_14135">приложение</span>, логика его работы, а также пользовательский интерфейс.</p><p id="36073ec8_14136">В папке <code class="code" id="36073ec8_14137">res</code>, находятся различные <span class="control" id="36073ec8_14138">статические</span> файлы - текст для компонентов, изображения, шрифты, вёрстка статичных компонентов (актуально для приложений на <code class="code" id="36073ec8_14139">view</code>).</p><p id="36073ec8_14140"><code class="code" id="36073ec8_14141">Gradle Scripts</code> - данная секция предназначена для работы <span class="emphasis" id="36073ec8_14142">системы автоматической сборки</span> <code class="code" id="36073ec8_14143">Gradle</code>.</p><p id="36073ec8_14144">Здесь мы можем описывать какие нам необходимы <span class="control" id="36073ec8_14145">зависимости</span> для нашего приложения, и в частности для нашего модуля, это позволяет построить многомодульное приложение, с конкретными разделениями зависимостей для работы каждого модуля.</p></section><section class="chapter"><h2 id="36073ec8_14146" data-toc="36073ec8_14146">Добавление зависимостей</h2><p id="36073ec8_14150"><p><code class="code" id="36073ec8_14151">implementation (&quot;com.squareup.retrofit2:retrofit:2.9.0&quot;)</code></p></li><li class="list__item" id="36073ec8_14152"><p><code class="code" id="36073ec8_14153">implementation (&quot;com.squareup.retrofit2:converter-gson:2.9.0&quot;)</code></p></li><li class="list__item" id="36073ec8_14154"><p><code class="code" id="36073ec8_14155">implementation(&quot;io.coil-kt:coil-compose:2.4.0&quot;)</code></p></li><li class="list__item" id="36073ec8_14156"><p><code class="code" id="36073ec8_14157">implementation (&quot;androidx.lifecycle:lifecycle-viewmodel-compose:2.6.2&quot;)</code></p><figure  id="36073ec8_14158"><img alt="img.png" title="img.png" src="images/4.png"  class="" width="2560" height="1391"/></figure></li></ul><p id="36073ec8_14159">Нажмём кнопку <code class="code" id="36073ec8_14160">Sync Now</code>, чтобы загрузить добавленные зависимости.</p></section><section
